
# RAG准确率优化 - 版本 2.1

## 📋 优化概述

本次优化主要针对两个典型错误进行优化，提高检索准确率和答案质量。

## 🎯 解决的问题

### 问题1：烟感型号混同模块
- **问题**：询问烟感型号时会包含模块型号
- **解决**：添加问题意图识别 + 检索结果过滤

### 问题2：EST3容量检索错误
- **问题**：搜索"EST3容量"时会找到单个模块容量而非系统整体容量
- **解决**：意图分析 + 搜索关键词优化

## ✨ 实现的功能

### 1. 问题意图分析 (`_analyze_and_optimize_question`)

| 功能描述：分析问题类型并优化搜索关键词

- **烟感问题**：添加"探测器 烟感 型号"关键词
- **模块问题**：添加"模块 型号"关键词
- **系统容量问题**：添加"系统整体容量 EST3 网络节点"关键词

### 2. 检索结果验证过滤 (`_filter_and_verify_docs`)

功能描述：根据问题类型过滤不相关的检索结果

- **烟感问题**：排除只包含模块型号的内容
- **模块问题**：排除只包含烟感型号的内容
- **系统容量问题**：优先选择包含系统相关关键词的内容

### 3. 增强提示词

更新提示词增加分类验证要求：

```
【分类验证要求】
- 如果问题涉及设备型号：仔细区分探测器/烟感型号 vs 模块/其他设备型号
- 如果问题涉及容量：明确区分系统整体容量 vs 单个设备容量 vs 回路容量
- 如果搜索结果中包含多种类型的信息，只保留与问题直接相关的部分
```
