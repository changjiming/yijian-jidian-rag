
# 📚 一建机电教材 RAG 问答系统 - 使用说明

## 🚀 快速开始

### 方案一：直接运行（推荐）

1. 双击 **启动程序.bat**
2. 程序会自动检查Ollama状态
3. 上传您的PDF教材
4. 开始问答！

### 方案二：打包成EXE

1. 双击 **打包程序.bat**
2. 等待打包完成
3. 在 `dist` 目录找到 **一建机电RAG.exe**

---

## 📋 系统要求

- **内存**: 8GB 以上（推荐 12GB+）
- **磁盘空间**: 至少 8GB 可用空间
- **操作系统**: Windows 10 或 11

---

## 📦 依赖安装

首次运行前，请安装Python依赖：

```bash
cd "d:\Claude Code\一建机电RAG"
pip install -r requirements.txt
```

---

## 🎯 使用步骤

### 1. 确保Ollama正在运行

```bash
ollama serve
```

### 2. 下载模型（如果还没有）

```bash
ollama pull qwen2.5:7b
ollama pull mxbai-embed-large
```

### 3. 启动程序

双击 **启动程序.bat** 或运行：

```bash
python desktop_app.py
```

### 4. 上传教材

- 点击「📖 上传教材」按钮
- 选择您的PDF教材文件
- 教材会自动保存到 `data/uploads` 目录

### 5. 开始问答

- 在左侧列表选择教材文件
- 在下方输入框输入问题
- 按回车或点击「发送」

---

## 🔧 故障排除

### 问题：Ollama未启动

**解决方法**：打开终端运行 `ollama serve`

### 问题：缺少模型

**解决方法**：
```bash
ollama pull qwen2.5:7b
ollama pull mxbai-embed-large
```

### 问题：界面显示乱码

**解决方法**：确保系统语言设置为中文

### 问题：响应很慢

**说明**：集成显卡使用CPU推理，首次响应可能需要10-30秒，这是正常的！
