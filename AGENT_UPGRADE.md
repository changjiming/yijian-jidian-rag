
# 一建机电 RAG - Agent 模式升级说明

## 新增功能

### 1. Agent 模式
- 🤖 新增 "Agent 模式" 按钮，可以切换普通 RAG 模式和 Agent 模式
- Agent 模式下，系统会先思考，然后决定是否需要使用工具

### 2. 思考链显示
- 🧠 在 Agent 模式下，会显示完整的思考过程
- 包括：思考步骤、工具使用、工具结果等
- 支持彩色标签区分不同类型的信息

### 3. 工具系统
- `search_textbook`: 搜索教材相关内容
- `get_page_content`: 获取指定页码的内容
- 工具注册系统，可方便扩展

## 使用方法

### 启动程序
```bash
cd "d:\Claude Code\一建机电RAG"
python desktop_app.py
```

### 切换模式
1. 点击顶部 "🤖 Agent 模式" 按钮
2. 按钮点击后，会出现 "🧠 Agent 思考链" 面板
3. 再次点击按钮可切换回普通模式

### Agent 模式工作流程
1. **思考阶段**: Agent 分析问题，决定是否需要工具
2. **工具阶段**: 如果需要，调用相关工具搜索资料
3. **回答阶段**: 基于搜索结果给出完整回答

## Agent 核心组件

### Agent 类
- 状态管理 (思考、使用工具、回答)
- 工具注册和调用
- 记忆系统 (短期、长期)
- 思考链记录

### AgentTool 类
- 工具基类
- 支持自定义工具扩展

### AgentMemory 类
- 短期记忆（最近对话）
- 长期记忆（历史对话）
- 自动记忆管理
